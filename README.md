# About

Adds a custom [Godot](https://godotengine.org/) Container that will arrange its children in a specific number of columns. Each column will have (as far as possible) the same number of elements.

This implementation is extremely barebones, but it can serve as a starting point for your own project

# Usage

Add the script to a Godot project and the class `ColumnLayoutContainer` will become available. It can be used like any other container, although (as stated above) there is a lot of functionality missing.
